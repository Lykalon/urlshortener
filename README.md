Тестовое задание.

Реализовать:
Сервис , который должен предоставлять API по созданию сокращенных ссылок формата:
- Ссылка должна быть уникальной и на один уникальный URL должна ссылаться одна уникальная ссылка
- Длина ссылки 10 символов
- Ссылка состоит из символов латинского алфавита в верхнем и нижнем регистрах, цифр и символа нижнего подчеркивания (_)

Запросы по http:
- POST - Сохраняет оригинальный URL и возвращает сокращенный
- GET - Принимает сохраненный URL т возвращает оригинальный

Условия для завершения задания:
- Сервис распространяется в виде Docker-образа
- В качестве хранилища можно использовать две реализации:
    1 - PostgreSQL
    2 - Локальное хранилище
- Покрыть функционал UNIT-тестами
